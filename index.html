Formula Mifflin-St Jeor
La formula piu accurata per il calcolo del metabolismo basale negli adulti.

Maschio: BMR = (10 x kg) + (6.25 x cm) - (5 x eta) + 5
Femmina: BMR = (10 x kg) + (6.25 x cm) - (5 x eta) - 161

TDEE = BMR x fattore attivita
